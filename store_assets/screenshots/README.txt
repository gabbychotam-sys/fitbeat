# Screenshots Guide
# ==================

## Option 1: From Simulator (Recommended for quick start)

Visit the simulator at:
https://fitness-goals-30.preview.emergentagent.com

Take screenshots of:
1. Main display (with some progress showing)
2. Settings screen
3. Goal picker
4. HR menu
5. Color menu

## Option 2: From Your Garmin Watch (Best quality!)

On your Fenix 8 Solar:
1. Press and hold LIGHT button + START button together
2. The screen will flash = screenshot taken!
3. Connect watch to computer via USB
4. Find screenshots in: GARMIN/SCRN/ folder
5. Copy to this screenshots folder

## Recommended Screenshots:

1. screenshot_main.png - Main watch face with time, distance, HR
2. screenshot_settings.png - Settings menu
3. screenshot_goal.png - Goal picker with arrows
4. screenshot_hr.png - Heart rate menu
5. screenshot_colors.png - Color selection menu

## Image Requirements for Store:

- Format: PNG
- Size: Garmin recommends device-native resolution
- For Fenix 8 Solar 51mm: 280x280 pixels
- Show the app in action!
